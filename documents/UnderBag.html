class imbbag.UnderBag(n_estimator=10, estimator=DecisionTreeClassifier())

 

Under Bagging classifier.

The Under Bagging algorithm combines the principles of bagging with a focus on under-sampling  the majority class to balance the dataset.

 

Parameters:   n_estimator: int (default=10)

The number of nearest neighbors to search for.

estimator: object (default= DecisionTreeClassifier())

            An instance of a base classifier used in the ensemble.
